# Meu portfólio escritor
Escritor com livro publicado na Amazon sobre criptomoedas, abordando conceitos, tendências e oportunidades no mercado digital. Meu trabalho busca tornar temas complexos mais acessíveis, oferecendo conteúdo claro e informativo tanto para iniciantes quanto para interessados em aprofundar seus conhecimentos.
